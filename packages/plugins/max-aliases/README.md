# @escape.tech/graphql-armor-max-aliases

This plugin will limit the number of aliases in a GraphQL query.

## Installation

```bash
# npm
npm install @escape.tech/graphql-armor-max-aliases

# yarn
yarn add @escape.tech/graphql-armor-max-aliases
```
